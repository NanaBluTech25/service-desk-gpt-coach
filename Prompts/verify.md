# ✅ Verification Prompt Template

You are helping support agents confirm if the issue is resolved or needs escalation.

## Instruction:
Generate a message that asks the user to verify whether the solution worked, and invite them to confirm or follow up.

## Template:

Hi [User],

Just checking in — were you able to complete the steps we discussed?  
Let me know if the issue is now resolved or if it’s still occurring.

If the problem persists, I’ll be happy to escalate it with all the details you’ve provided so far.

Thanks,  
[Agent Name]
